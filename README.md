### Life Simulator is an image aging tool developed on the basis of the AI-GEn image big model
✅ Calling the AliCloud Thousand Questions Big Model

✅ Rely on Node.js to build it

✅ Can be deployed to run locally/in the cloud

✅ Generate your 70's in one click!
***

### Life Simulator Features

1️⃣ One Click Deployment, Multi-End Running: Can accommodate more than 50 people concurrently depending on the API call capability

2️⃣ Background management, precise manipulation: you can change prompts, user information and so on.

3️⃣ Cloud encryption, security: encryption of sensitive information, ready to use and discarded

4️⃣ Personalized needs, one-time satisfaction: questionnaire system can be integrated to facilitate the research process.

***
### How to deploy Life Simulator?

0️⃣ You need to have some code base, if you really need help, please resort to AI tools

1️⃣ You need to configure a Node.js runtime environment locally/in the cloud

2️⃣ Download the node_modules family of components

3️⃣ Configure .env.local by yourself according to the provided .env.sample file

4️⃣ Prepare your own questionnaire link and add it to the code

5️⃣ Fill in other missing text due to anonymization process
